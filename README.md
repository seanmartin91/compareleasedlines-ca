# Compare Leased Lines CA

## Project Documentation

### Overview
The Compare Leased Lines CA platform allows users to compare various leased line providers and their offerings, making it easier to select the best option for their needs.

### Features
- **Comparison Tool**: Easily compare different leased line providers and their offerings.
- **Provider Ratings**: View ratings and reviews for each provider.
- **Custom Filters**: Use filters to narrow down to the best options based on speed, price, and availability.
- **User-Friendly Interface**: Intuitive design for seamless navigation.
- **Real-Time Updates**: Get the latest information on leased line services.

### Tech Stack
- **Frontend**: React.js, Redux, CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **APIs**: RESTful APIs for third-party provider data
- **Hosting**: AWS

### Getting Started
To get a local copy up and running, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/compareleasedlines-ca.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd compareleasedlines-ca
   ```

3. **Install the dependencies**:
   ```bash
   npm install
   ```

4. **Run the application**:
   ```bash
   npm start
   ```
